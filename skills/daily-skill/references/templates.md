# Daily Note Templates

## Standard Weekday Template

```markdown
---
date: {{DATE}}
tags:
  - daily
type: daily
---

# {{DATE}}

## Carry Forward
<!-- Auto-populated from yesterday's unfinished items -->

## Plan
-

## Notes

## Blockers

## End of Day
-
```

```markdown

<!-- What's the main goal for this week? -->

```
